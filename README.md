# 🚀 DV Solutions | Portafolio Full Stack

Bienvenido a mi portafolio 2026. Soy **Daniela Villarroel**, desarrolladora web full stack JavaScript enfocada en crear experiencias digitales funcionales, visualmente atractivas y 100% personalizadas.

## 🎯 Sobre mí

Me especializo en desarrollo full stack con **JavaScript**, creando soluciones donde:
- El **frontend** comunica con fuerza y claridad
- El **backend** responde con orden, seguridad y escalabilidad
- Cada proyecto tiene una **identidad visual clara** y propósito

Mi enfoque une diseño, lógica y detalle, pensando siempre en la experiencia real de quien navega y en que cada pieza del producto tenga sentido.

## 🚀 Cómo usar este portafolio

1. **Clona el repositorio:**
   ```bash
   git clone [tu-repo-url]
   cd Mi-Portafolio
   ```

2. **Abre en tu navegador:**
   ```bash
   # Simplemente abre el archivo index.html en tu navegador
   # o usa un live server
   ```

3. **Personaliza:**
   - Edita `script.js` para agregar más proyectos
   - Modifica `styles.css` para cambiar colores y estilos
   - Actualiza información personal en `index.html`
